# Docs Automation Label Taxonomy

<!-- cspell:ignore BFD4F2 FBCA04 D93F0B D4C5F9 E99695 F9D0C4 C2E0C6 -->

This file is the source of truth for labels used by the docs automation workflow.

## Issue Lifecycle Labels

| Label | Description | Color |
| --- | --- | --- |
| `needs-triage` | New docs request waiting for triage. | `#BFD4F2` |
| `needs-info` | Missing required fields; request cannot proceed. | `#FBCA04` |
| `agent:ready` | Request is triaged and ready for automation execution. | `#0E8A16` |
| `agent:running` | Automation workflow is currently running. | `#1D76DB` |
| `agent:blocked` | Automation is blocked and needs maintainer/requester input. | `#D93F0B` |
| `agent:pr-open` | Automation opened a PR for the request. | `#5319E7` |

## PR Lifecycle Labels

| Label | Description | Color |
| --- | --- | --- |
| `agent:generated` | PR was created by automation. | `#0052CC` |
| `agent:patch` | Agent may push follow-up commits to this PR branch. | `#C2E0C6` |
| `ui:accept-baseline` | Baseline visual updates are explicitly approved for this PR. | `#F9D0C4` |
| `agent:large` | Reviewer-approved exception for large docs diff scope. | `#E99695` |
| `docs:not-needed` | Explicitly approved reason to skip docs updates. | `#D4C5F9` |

## Sync Command

Apply or update all labels with GitHub CLI:

```bash
gh label create "needs-triage" --color BFD4F2 --description "New docs request waiting for triage" --force
gh label create "needs-info" --color FBCA04 --description "Missing required details; cannot proceed" --force
gh label create "agent:ready" --color 0E8A16 --description "Triaged and ready for docs automation" --force
gh label create "agent:running" --color 1D76DB --description "Docs automation workflow is running" --force
gh label create "agent:blocked" --color D93F0B --description "Automation blocked pending more input" --force
gh label create "agent:pr-open" --color 5319E7 --description "Automation opened a pull request" --force
gh label create "agent:generated" --color 0052CC --description "Pull request generated by automation" --force
gh label create "agent:patch" --color C2E0C6 --description "Agent may push additional patch commits" --force
gh label create "ui:accept-baseline" --color F9D0C4 --description "Approve visual regression baseline updates" --force
gh label create "agent:large" --color E99695 --description "Approved large-scope automated docs change" --force
gh label create "docs:not-needed" --color D4C5F9 --description "Approved to skip docs updates for this change" --force
```
