# Reviewer Rubric: Style and Terminology

## Blocking Findings

- terminology violates `agent/GLOSSARY.md`
- tone violates style constraints (hype, unclear wording, non-technical marketing)
- page missing required frontmatter

## Non-Blocking Suggestions

- improve concise active-voice phrasing
- reduce sentence length and passive constructions
- improve component usage (callouts/cards/switcher) for clarity

## Review Output Format

- `Blocking issues`
- `Non-blocking suggestions`
- `Suggested patch snippets` (small, optional)
- `Confidence: high|med|low`
