# Reviewer Rubric: Information Architecture

## Blocking Findings

- content placed in wrong top-level section
- major duplication of canonical content across pages
- new page introduces navigation ambiguity

## Non-Blocking Suggestions

- improve cross-linking between related pages
- refine titles for discoverability
- recommend consolidation when overlap is moderate

## Review Output Format

- `Blocking issues`
- `Non-blocking suggestions`
- `Suggested patch snippets` (small, optional)
- `Confidence: high|med|low`
