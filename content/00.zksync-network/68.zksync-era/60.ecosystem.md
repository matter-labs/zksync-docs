This page lists third-party tools and services that you can use with ZKsync
Era.

The list is intentionally selective. Always verify current support and
functionality before relying on any external service.

## Bridges

- [Portal Bridge](https://bridge.zksync.io/): Official bridge for moving assets
  between Ethereum and ZKsync Era.
- [Across Bridge](https://app.across.to/bridge): Cross-chain bridge that
  supports transfers involving ZKsync Era.
- [Layerswap](https://layerswap.io/app): Asset transfer service that supports
  deposits to ZKsync Era.
- [Orbiter Finance](https://www.orbiter.finance/bridge/Ethereum/ZKSyncEra?token=ETH):
  Bridge page with ZKsync Era selected as the destination network.
- [Owlto Finance](https://owlto.finance/): Bridge service used for transfers to
  and from ZKsync Era.

## Block explorers

### Official explorer

The official ZKsync Era Block Explorer is the main interface for inspecting
transactions, blocks, batches, wallets, tokens, and smart contracts on the
network.
It also provides the standard explorer API and contract verification endpoints
used in many ZKsync Era workflows.

- Mainnet: [ZKsync Era Block Explorer](%%zk_mainnet_block_explorer_url%%)
- Testnet: [ZKsync Era Testnet Block Explorer](%%zk_testnet_block_explorer_url%%)
- Mainnet API Swagger: [block-explorer-api.mainnet.zksync.io/docs](https://block-explorer-api.mainnet.zksync.io/docs)
- Testnet API Swagger: [block-explorer-api.sepolia.zksync.dev/docs](https://block-explorer-api.sepolia.zksync.dev/docs)
- Mainnet verification API: `%%zk_mainnet_contract_verification_url%%`
