Welcome to the Quickstart!

This tutorial will help you to learn how to write, deploy, and interact with smart contracts on a ZKsync chain.

In the first step, you will build and deploy a simple token contract.
Next, you will test the contract by running a script to interact with it.
Finally, you will create a frontend for the contract.

This section is designed for developers new to ZKsync and gives you the options of deploying to either a local node or a testnet
using the most popular smart contract development frameworks.

It will help you get familiar with the ZKsync smart contract development and deployment process using different tools.

In this section you will learn how to:

:check-icon Get setup with either the testnet or a local node.

:check-icon Build an ERC-20 token contract.

:check-icon Deploy the smart contract.

:check-icon Interact with the contract using a script.

:check-icon Interact with the contract in a simple frontend.
