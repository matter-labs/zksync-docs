The ZK Stack is open-source software available under permissive licenses. Organizations can freely use, modify, and deploy ZK Stack components.

## License Terms

ZK Stack components are dual-licensed under:

- **MIT License**
- **Apache License 2.0**

Choose either license based on your project requirements. Both licenses permit commercial use, modification, and distribution.

## What's Included

The following components are covered under the open-source licenses:

- ZKsync Era node and sequencer
- Prover infrastructure
- Smart contract system
- Standard RPC API
- Block explorer (open-source version)
- JavaScript, Go, Swift, and Rust SDKs

## Source Code

Access the source code on GitHub:

- [Era Contracts](https://github.com/matter-labs/era-contracts)
